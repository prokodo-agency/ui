Quick Start für Next.js und Vite in 3 Schritten

Integriere einen AIC-ready, High-Performance Komponenten-Stack in nur wenigen Schritten in deine React-App.

1Installieren

Füge die Library in dein React-Projekt ein und starte direkt mit produktionsreifen UI-Bausteinen.

$pnpm add @prokodo/ui
2Theme importieren

Aktiviere token-basiertes Theming für konsistente Light- und Dark-Experiences über alle Komponenten.

import '@prokodo/ui/theme.css'
3Verwenden

Importiere typisierte, tree-shakable Komponenten, optimiert für Performance und Accessibility.

import { Button } from '@prokodo/ui/button'
