# List

Flexible list component rendering an array of items as styled `<ul>/<ol>` with support for icons, actions, cards layout, and custom item render functions.

## When to use

Use `List` for displaying a set of related items — feature lists, navigation menus, settings options, or content feeds. For tabular data use `Table`.

## Accessibility (WCAG 2.2)

- Renders a semantic `<ul>` or `<ol>` with `<li>` elements
- Interactive items (links, buttons) carry appropriate roles and focus styles
- Icon-only items require `aria-label` on the icon or a visually hidden text label
- List `aria-label` is recommended when multiple lists appear on the same page

## ✅ Do

- Use ordered lists (`ol`) when sequence matters (steps, rankings)
- Provide `aria-label` on the list element when context is not clear from surrounding headings
- Group related lists with a visible heading

## ❌ Don't

- Don't use `List` purely for visual spacing — use layout components
- Don't place block-level elements (headings, sections) inside list items
- Don't mix interactive and non-interactive items in the same list without visual distinction

## API notes

- `color` controls the clickable-item colour modifier applied as a BEM modifier (`--is-clickable--{color}`)
- `type` can be `"default"` (plain list with optional icons) or `"card"` (each item wrapped in a `Card`)
- Per-item `color` overrides the list-level `color` for that item

## Design Tokens

Key `--pk-*` tokens used by this component:

| Token                       | Purpose                                      |
| --------------------------- | -------------------------------------------- |
| `--pk-space-xs`             | Item vertical margin, icon offset            |
| `--pk-space-sm`             | Icon wrapper padding, card title sub-padding |
| `--pk-space-md`             | Card item padding                            |
| `--pk-space-2xl`            | Item content width calculation               |
| `--pk-color-muted`          | Default item and description text color      |
| `--pk-color-brand`          | Primary hover / active color                 |
| `--pk-color-accent`         | Secondary color                              |
| `--pk-color-surface-raised` | Icon circle background                       |
| `--pk-palette-white`        | White item text                              |
