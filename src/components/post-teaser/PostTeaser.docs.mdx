# PostTeaser

A featured blog-post card displaying a hero image, headline, excerpt, reading time, publish date, and structured SEO metadata. Used in listing pages and editorial feeds.

## When to use

Use `PostTeaser` as the primary card for editorial content previews in a grid or feed layout. For compact list rows use `PostItem`; for a sidebar summary widget use `PostWidget`.

## Accessibility (WCAG 2.2)

- The headline link has a unique, descriptive accessible name that matches the post title
- Reading time and publish date are surfaced in `<time>` elements
- Structured data (`structuredData`) is injected via a `<script type="application/ld+json">` and does not affect DOM focus

## ‚úÖ Do

- Always provide `locale` so dates are formatted in the reader's language
- Include `structuredData` for SEO-rich snippets in search results

## Design Tokens

Key `--pk-*` tokens used by this component:

| Token                         | Purpose                                                   |
| ----------------------------- | --------------------------------------------------------- |
| `--pk-space-sm`               | Caption offset, read-count wrapper gap                    |
| `--pk-space-lg`               | Card content/footer padding, meta overlay offset          |
| `--pk-color-muted`            | Date text color                                           |
| `--pk-color-brand`            | Hovered link color                                        |
| `--pk-color-brand/accent/...` | Colored border, background tint, and shadow (via `color`) |

## Color

The `color` prop provides a shared visual accent across the entire teaser card:

- **Card**: receives the same `color` value, enabling the colored `border`,
  background tint, and ambient `box-shadow` that keep blog components
  consistent with inputs, buttons, and other design-system elements.
- **Headline**: uses `color` as its fallback when `title.color` is not set.

Available values: `inherit` ¬∑ `primary` ¬∑ `secondary` ¬∑ `success` ¬∑ `warning` ¬∑ `error` ¬∑ `info` ¬∑ `white`
