# PostWidgetCarousel

An auto-playing carousel of post cards. Extends `PostWidget` with swipeable slides, autoplay controls, and locale-aware date formatting.

## When to use

Use `PostWidgetCarousel` when you need to surface several posts in limited horizontal space — e.g. a "Latest articles" strip on a landing page. For static lists prefer `PostWidget`.

## Accessibility (WCAG 2.2)

- The carousel exposes `role="region"` with an `aria-label`
- Autoplay is paused on focus and on hover per WCAG 2.2 SC 2.2.2
- Previous/Next controls have accessible labels and are keyboard-reachable
- Each slide is announced when it becomes active via `aria-live="polite"`

### Keyboard behavior

- **Tab**: move into the carousel controls
- **←** / **→**: navigate slides when a control is focused
- **Space**: pause / resume autoplay

## ✅ Do

- Provide `locale` so dates render in the user's language
- Ensure every item has a unique, descriptive title as the accessible name

## ❌ Don't

- Don't set very short autoplay intervals (< 4 s) — it prevents reading
- Don't hide navigation arrows on touch devices; swipe is not universally available

## Design Tokens

Key `--pk-*` tokens used by this component:

| Token                         | Purpose                                                   |
| ----------------------------- | --------------------------------------------------------- |
| `--pk-space-sm`               | Nav button right position, button gap                     |
| `--pk-space-md`               | Nav button bottom position, headline top margin           |
| `--pk-space-lg`               | Carousel wrapper bottom padding                           |
| `--pk-space-xl`               | Carousel section top margin                               |
| `--pk-radius-md`              | Slide image border-radius                                 |
| `--pk-color-brand/accent/...` | Colored border, background tint, and shadow (via `color`) |

## Color

The `color` prop drives a unified visual theme across all nested elements,
identical in behavior to `PostWidget` and consistent with the design system:

- **Card** (outer shell): colored `border`, background tint, and `box-shadow`.
- **Title Headline**: inherits `color` unless `title.color` overrides it.
- **Item Headlines**: inherit `color` unless the item's `title.color` overrides.
- **Prev / Next Buttons**: receive `color` as their button color (overridable
  via `componentsProps.prevButton` / `componentsProps.nextButton`).

Available values: `inherit` · `primary` · `secondary` · `success` · `warning` · `error` · `info` · `white`
