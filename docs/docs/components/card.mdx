# Card

`Card` is a multipurpose container that groups related content. It supports multiple visual variants (`panel`, `list-item`), optional shadow, animated entrance, and a clickable mode via `redirect` or `onClick`.


---

## Overview

```tsx
import { Card } from '@prokodo/ui/card'

<Card variant="panel" enableShadow animated={false}>
  <h2>Title</h2>
  <p>Card content goes here.</p>
</Card>
```
