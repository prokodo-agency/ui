# Icon

`Icon` renders SVG icons from the prokodo icon set. In production builds (and inside Docusaurus), icons are loaded from the jsDelivr CDN via `process.env.MODE = 'production'`.

---

## Overview

```tsx
import { Icon } from '@prokodo/ui/icon'

<Icon name="arrow-right" size={24} color="primary" />
```
