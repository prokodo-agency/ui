# Map

`Map` renders an interactive map. Markers, clusters, polygons, and custom info windows are configured via props. The underlying library (Google Maps / Leaflet) is abstracted behind a consistent API.

---

## Overview

```tsx
import { Map } from '@prokodo/ui/map'

<Map
  center={{ lat: 52.52, lng: 13.405 }}
  zoom={12}
  markers={[{ position: { lat: 52.52, lng: 13.405 }, title: 'Berlin HQ' }]}
/>
```
