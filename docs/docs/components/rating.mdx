# Rating

`Rating` renders a star-based rating input. Supports interactive selection, read-only display, half-star precision, and accessible ARIA role `slider`.


---

## Overview

```tsx
import { Rating } from '@prokodo/ui/rating'

<Rating
  name="product-rating"
  value={rating}
  onChange={setRating}
  max={5}
/>
```
