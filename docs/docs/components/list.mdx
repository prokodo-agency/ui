# List

`List` renders semantic `<ul>` or `<ol>` lists with optional icon bullets and support for nested sub-lists.

---

## Overview

```tsx
import { List } from '@prokodo/ui/list'

<List
  items={[
    { text: 'First item' },
    { text: 'Second item', children: [{ text: 'Nested item' }] },
  ]}
/>
```
