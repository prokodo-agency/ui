# Calendly

`Calendly` embeds a Calendly scheduling widget directly in the page. Supports inline and popup modes. The iframe is lazy-loaded for performance.


---

## Overview

```tsx
import { Calendly } from '@prokodo/ui/calendly'

<Calendly url="https://calendly.com/your-org/30min" mode="inline" />
```
