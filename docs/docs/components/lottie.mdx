# Lottie

`Lottie` renders Lottie JSON animations. Supports autoplay, loop, and programmatic play/pause control. Animations are loaded from a URL or inline JSON data.

---

## Overview

```tsx
import { Lottie } from '@prokodo/ui/lottie'

<Lottie src="/animations/hero.json" loop autoplay />
```
