# PostWidget

`PostWidget` renders a compact sidebar panel listing recent or related posts. Each entry shows a thumbnail, title, and date.


---

## Overview

```tsx
import { PostWidget } from '@prokodo/ui/post-widget'

<PostWidget
  title="Recent posts"
  posts={latestPosts}
/>
```
