# RichText

`RichText` renders sanitised HTML or Portable Text / structured content from a CMS. Applies prokodo typography styles to all child elements (`<h2>`‚Äì`<h6>`, `<p>`, `<a>`, `<ul>`, `<ol>`, `<blockquote>`).


---

## Overview

```tsx
import { RichText } from '@prokodo/ui/rich-text'

<RichText html={article.content} />
```


No separate `/client` or `/lazy` import selection is required in consumer code.
