# Drawer

`Drawer` renders an off-canvas side panel that slides in from the left, right, top, or bottom. Focus is trapped while open and released on close.


---

## Overview

```tsx
import { Drawer } from '@prokodo/ui/drawer'

<Drawer
  open={menuOpen}
  placement="right"
  onClose={() => setMenuOpen(false)}
>
  <nav>...</nav>
</Drawer>
```
