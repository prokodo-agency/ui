# Button

The `Button` component is the core interactive element in `@prokodo/ui`. It supports text labels, icon-only mode, loading states, full-width layout, and seamless redirect (SSR-safe link conversion).


---

## Overview

```tsx
import { Button } from '@prokodo/ui/button'

<Button title="Click Me" color="primary" variant="contained" onClick={handleClick} />
```

## Props

`ButtonProps` is a discriminated union of `ButtonDefaultProps` (labeled button) and `ButtonIconProps` (icon-only button).

### `ButtonDefaultProps`

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| `title` | `string` | ‚Äî | ‚úÖ | Button label text (also used as fallback `aria-label`). |
| `variant` | `"contained" \| "outlined" \| "text"` | `"contained"` | ‚Äî | Visual style variant. |
| `color` | `"primary" \| "secondary" \| "success" \| "error" \| "info" \| "warning" \| "inherit"` | `"inherit"` | ‚Äî | Semantic color token. |
| `priority` | `boolean` | `false` | ‚Äî | Emphasize the button (higher visual weight). |
| `fullWidth` | `boolean` | `false` | ‚Äî | Stretch button to 100% of parent width. |
| `loading` | `boolean` | `false` | ‚Äî | Show loading spinner and disable interaction. |
| `disabled` | `boolean` | `false` | ‚Äî | Disable all interaction. |
| `iconProps` | `IconProps` | ‚Äî | ‚Äî | Icon to display left of (or alongside) the label. |
| `redirect` | `LinkProps` | ‚Äî | ‚Äî | Convert button into an SSR-safe link. |
| `aria-label` | `string` | `title` | ‚Äî | Override accessible label. |
| `ref` | `ButtonRef` | ‚Äî | ‚Äî | Ref to the underlying `<button>` element. |
| `contentClassName` | `string` | ‚Äî | ‚Äî | CSS class on the inner content wrapper. |
| `className` | `string` | ‚Äî | ‚Äî | CSS class on the root element. |
| `onClick` | `MouseEventHandler` | ‚Äî | ‚Äî | Click handler. |
| `image` | `ImageProps` | ‚Äî | ‚Äî | Background image (alternative to icon). |

### `ButtonIconProps`

Icon-only variant. Requires either `aria-label` (interactive) or `inert` (decorative).

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| `iconProps` | `IconProps` | ‚Äî | ‚úÖ | Icon to render. |
| `aria-label` | `string` | ‚Äî | ‚úÖ* | Accessible label for screen readers. |
| `inert` | `boolean` | ‚Äî | ‚úÖ* | Mark as decorative (no a11y label needed). |

\* Either `aria-label` or `inert` must be provided (not both).

## AIC Note

Use the standard import path in application code:

```ts
import { Button } from '@prokodo/ui/button'
```

No separate `/client` or `/lazy` import selection is required in consumer code.

AIC components also support a `priority` flag for critical above-the-fold elements.
This is most visible on `Image` (`loading="eager"` + `fetchpriority="high"`).
