# Grid

`Grid` renders a responsive CSS grid container. Column count and gap are configurable per breakpoint, making it easy to build multi-column page sections.

---

## Overview

```tsx
import { Grid } from '@prokodo/ui/grid'

<Grid cols={3} gap="lg">
  <Card>Item 1</Card>
  <Card>Item 2</Card>
  <Card>Item 3</Card>
</Grid>
```
