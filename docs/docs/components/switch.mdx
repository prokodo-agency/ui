# Switch

`Switch` renders a toggle switch (ARIA `role="switch"`) for binary on/off states. Visually distinct from `Checkbox` ‚Äî use `Switch` for settings and preference toggles.


---

## Overview

```tsx
import { Switch } from '@prokodo/ui/switch'

<Switch
  name="notifications"
  label="Email notifications"
  checked={enabled}
  onChange={(e) => setEnabled(e.target.checked)}
/>
```
