# Versions

## Current versions

| Version | Status    | React | Next.js |
| ------- | --------- | ----- | ------- |
| **1.x** | ✅ Active | ≥ 18  | ≥ 14    |
| **0.x** | ⛔ EOL    | ≥ 18  | ≥ 14    |

## Support policy

- **Active** — new features, bug fixes, and security updates
- **EOL** — no further updates; upgrade recommended

## Viewing older docs

Use the version switcher in the top navigation:

- **v1 (current)** opens the latest documentation
- **v0 (legacy)** links to the migration section (`v0 → v1`) because a standalone archived v0 docs snapshot is not published yet
