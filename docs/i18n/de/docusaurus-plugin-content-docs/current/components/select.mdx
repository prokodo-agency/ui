# Select

`Select` rendert ein barrierefreies Dropdown mit optionaler Suchfilterung, gruppierten Optionen, Multi-Select und asynchronem Laden externer Optionen.

```tsx

  name="country"
  label="Country"
  options={[
    { label: 'Germany', value: 'de' },
    { label: 'Austria', value: 'at' },
  ]}
  value={country}
  onChange={setCountry}
/>
```
